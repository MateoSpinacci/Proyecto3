# Restaurant Management System (Java)

This project is a console-based restaurant management system developed in Java, focused on practicing Object-Oriented Programming concepts.

## Features

* Customer management (regular, members, minors)
* Table assignment with capacity control
* Order creation with products and quantities
* Payment system with validation
* Discount system using interfaces
* Business logic handled through service layer

## Technologies & Concepts

* Java
* Object-Oriented Programming (OOP)
* Inheritance
* Polymorphism
* Abstraction
* Interfaces
* Arrays
* Basic system design (model/service structure)

## Project Structure

* `model` → core entities (Cliente, Pedido, Mesa, Producto, etc.)
* `service` → business logic (Servicio)
* `app` → main execution (Runner)

## Highlights

* Use of polymorphism for handling different product types
* Interface-based discount system
* Validation of real-world scenarios (table capacity, payment status, etc.)
* Separation of concerns between logic and data
